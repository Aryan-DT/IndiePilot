Badge Images Directory

This directory would contain small PNG badge images for:
- First Steps (first transaction logged)
- Week Warrior (7-day streak)
- Monthly Master (30-day streak)
- Saver (saved $100)
- Giver (shared $50)

Each badge would be a simple, colorful 64x64 PNG with:
- Clear icon representing the achievement
- Bright, engaging colors
- Teen-friendly design
- Consistent style across all badges

For the demo, these are placeholder files.
